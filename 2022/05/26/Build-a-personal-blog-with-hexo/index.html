Build a personal blog with hexo

2022-05-26- 2022-05-28
Hexo

Step One: Install node.js

Click to enter the official website. And choose the LTS(Long Term Support) version. Then start nodejs installer that you’ve just downloaded. The installation process is very simple, basically the “next” of the whole process. You can run the following commands in the terminal to check whether the installation is successful.

Step Two: Configure npm

  1. Create the new path for global modules and cache.

Otherwise, the global modules and cache will be stored in the default directory, maybe in your system drive.

  1. Configure with cmd commands. Note that run the terminal as Administrator.

  1. Environment variables.

    • Create a system variable.
    • Change the user variable.

  2. Install cnpm.

  1. Check whether the installation is successful.

If prompted, the script file cannot be run, use the following command to change the execution policy. (Run the terminal as Administrator)

Step Three: Install hexo

Step Four: Init blog directory

  1. Create a new directory for your blogs.

  2. Enter the created directory and initialize it through hexo.

Step Five: Deploy the blog to Github

  1. Install the hexo plugin of deployment.

  1. Create a new repository in Github.

  2. Modify the _config.yml.

  1. Use your git to generate a ssh key.
  1. Add deploy key for your Github repository.

Copy the content of id_rsa.pub to the input box of key.

  1. Generate static files and deploy to the remote sites.
